			i-7065D 基本协议说明

    'START----------------------------------------------------------START
    '设置工作参数协议
    '%AANN40CCFF[CheckSum](CR)   / TT 为模块类型，i-7065D 固定为 40
    'AA 地址，NN 新地址，CC 波特率：
    'Code      03   04   05   06   07    08    09    0A
    'Baud rate 1200 2400 4800 9600 19200 38400 57600 115200
    'FF: 00 无校验，40 累加和
    'Response:
    'Valid Command: !NN[CHKSUM](CR)
    'Invalid Command: ?NN[CHKSUM](CR)
    '注意：修改波特率和校验方式必须在 INIT 模式下，
    '但修改模块地址不必在 INIT 模式下
    'END--------------------------------------------------------------END


    'START----------------------------------------------------------START
    '查询模块 EEPROM 中的工作参数
    '$AA2
    '返回：!AATTCCFF, 错误：?AA
    'END--------------------------------------------------------------END


    'START----------------------------------------------------------START
    '控制单个输出开关
    '#AA1cDD
    '#为本协议的标志，AA为模块地址，c=0-7，DD开关数据，01为闭合，00为断开。
    '如#021001 表示将地址为02的模块的第0路闭合(DD=01)。
    '返回：>[CheckSum: 3E]
    'Valid command: >[CHKSUM](CR)
    'Invalid command: ?[CHKSUM](CR)
    'END--------------------------------------------------------------END


    'START----------------------------------------------------------START
    '控制所有输出开关
    '@AADD
    '@为本协议的标志，AA为模块地址，
    'DD为开关数据（范围：00-1F），1为闭合，0为断开。
    '返回：>[CheckSum: 3E], ?[CHKSUM](CR)
    'END--------------------------------------------------------------END


    'START----------------------------------------------------------START
    '读开关状态
    '$AA6
    '$为本协议的标志，AA为模块地址，如此即可读取当前的开关状态。
    '返回：!DDDD00，第一个DD为输出开关状态（1为闭合），第二个为输入开关状态（1为断开）；
    '错误：?AA
    'END--------------------------------------------------------------END
